# IQ Battery Design Matrix

**Date:** 2026-03-07
**Question:** If people talk about IQ as "raw talent," what do the actual batteries do with time limits, finite item pools, adaptive routing, and progressively harder end items?

## Short Answer

No mainstream battery in this note is literally "unlimited items."

What changes across batteries is:

1. whether speed is directly rewarded
2. whether almost everyone finishes in time
3. whether the test gets harder until a ceiling / discontinue point
4. whether many examinees never even see the hardest items

So the honest description is not "IQ measures raw talent." It is "IQ-like batteries measure performance on a finite, normed, design-chosen surface." [INFERENCE]

## Three Different Meanings Of "Exhaust Items"

People collapse three different things:

1. **Clock exhaustion:** you run out of time before finishing the administered items.
2. **Form exhaustion:** you reach the last item in the administered form.
3. **Difficulty exhaustion:** the battery keeps getting harder until you hit a ceiling, discontinue rule, or adaptive stopping point.

Those are not the same event. [INFERENCE]

## Battery Notes

### `WAIS-IV / WAIS-5`

Pearson's public WAIS-IV Q&A says:

- `Block Design` keeps time bonus points because faster performance helps discriminate higher ability.
- `Visual Puzzles` and `Figure Weights` have strict time limits.
- `Matrix Reasoning` has a 30-second guideline rather than the same kind of hard speed bonus.

Pearson's `WAIS-5` page also says gifted examinees may meet discontinue criteria late or not at all, which is why special gifted start points were added. [SOURCE: https://www.pearsonassessments.com/store/en/usd/p/100000392.html; https://www.pearsonassessments.com/store/en/usd/p/P100071002.html]

So the adult Wechsler family is plainly not "unlimited untimed raw talent." It is a mixed battery with explicit speed-sensitive components. [INFERENCE]

### `Woodcock-Johnson IV`

Riverside's examiner workbook says many `WJ IV` tests use basal and ceiling rules to avoid administering items that are too easy or too hard. It also states that if a ceiling is not reached, administration continues to the last item. The same workbook identifies explicitly timed fluency tests such as `Sentence Reading Fluency` and `Math Facts Fluency`. [SOURCE: https://info.riversideinsights.com/hubfs/WJ%20IV%20University%20Training%20Program%20Matls%202022/ACH%20Materials/Module%201/Tests%20of%20Achievement%20Examiner%20Training%20Workbook.pdf]

That makes `WJ IV` closer to a power battery than `WAIS`, but still not unlimited. [INFERENCE]

### `Stanford-Binet 5`

Riverside's public product page explicitly lists:

- `Item Book 1 (Routing Subtests)`
- `Item Book 2 (Nonverbal Subtests)`
- `Item Book 3 (Verbal Subtests)`

That is enough to verify routing structure publicly. A distributor page reports about 5 minutes for each of 10 subtests, but I am treating the exact timing detail as lower-confidence because it is not the vendor's own page. [SOURCE: https://store.riversideinsights.com/p/stanford-binet-intelligence-scales-sb5/; https://www.mindresources.com/psychology/051950]

So `SB5` is a finite routed battery. I did not find a public exact completion-rate table in this pass. [INFERENCE]

### `Raven's 2`

The `WPS` product page for `Raven's 2` reports administration times of:

- `30` minutes for ages `4-9`
- `45` minutes for ages `9-90`

Pearson's `Q-global` page says the test has paper, digital long-form, and digital short-form options. [SOURCE: https://www.wpspublish.com/ravens2-progressive-matrices-clinical-edition; https://qglobal.pearsonclinical.com/qg/static/Product/es/Ravens2/Ravens2_Enter_Scores.htm]

So the current commercial Raven family is not an unlimited open-ended matrix stream. It is a finite, normed form with progressive difficulty and published administration times. [INFERENCE]

### `CAT-ASVAB`

Official ASVAB documentation says:

- power-test time limits were set so `>95%` should finish without rushing
- actual completion is `>98%` on each adaptive test

The test adapts item difficulty and stops after a fixed number of items or when the time limit is reached. [SOURCE: https://www.officialasvab.com/applicants/cat-asvab/; https://www.officialasvab.com/docs/asvab_techbulletin_1.pdf; https://www.nlsinfo.org/content/cohorts/nlsy97/other-documentation/codebook-supplement/appendix-10-cat-asvab-scores]

This is the cleanest public example here of a battery that is finite and adaptive while still being designed to behave like a power test rather than a speed trap. [INFERENCE]

### `CogAT`

Riverside's official timing page says `CogAT` Levels `5/6` through `8` are untimed and use only approximate working times, while upper levels are timed. [SOURCE: https://onlinehelp.riversideinsights.com/Help/Elevate2SHP/Topics/011_Plan_Your_CogAT_Testing_Program/CogAT_Testing_Times.htm]

So if someone wants an existence proof of a reasoning battery that deliberately deweights speed, this is one. It is still finite, though. [INFERENCE]

### `PISA`

`PISA` is not a clinical IQ battery, but it is a useful comparison because people often talk about it as if it were an IQ-like "talent" surface. OECD reports that the vast majority of students finished within the allotted time. Each student sees only a rotated subset of the total pool, and scoring is accuracy-based rather than speed-bonus-based. [SOURCE: https://www.oecd.org/en/publications/pisa-2018-results-volume-i_5f07c754-en/full-report/component-25.html]

In this repo's local `PISA 2018` passes, broad female-slower timing survives multiple matching upgrades, but it does not line up tightly with the localized score-residual ordering. [SOURCE: `research/iq-sex-differences-pisa2018-time-dif-theta.md`; `research/iq-sex-differences-pisa2018-dif-theta-logit.md`]

## What This Means For The "Raw Talent" Critique

If a battery is:

- generously timed
- accuracy-scored
- and finite

then it is deweighting speed. That is a construct choice.

If a battery has:

- hard section limits
- time bonuses
- or explicitly speeded subtests

then it is folding speed into the measured trait. That is also a construct choice.

Neither design is automatically "dishonest." The dishonest move is talking as if both designs are measuring the same thing. [INFERENCE]

## Raven Sex-Difference Snapshot

For `Raven` specifically, there is **not one stable male-female number** that survives all versions and samples.

The cleanest current summary is:

1. older mixed-sample meta-analytic work on adult `Raven` often reports a **small male advantage**, around `d = 0.31` to `0.33`
2. representative or more tightly modeled `Raven` studies can be **near zero**
3. so `Raven` is informative, but it does **not** settle the whole sex-difference question by itself

[SOURCE: https://www.sciencedirect.com/science/article/abs/pii/S0191886920301483; `research/iq-sex-differences-test-construction.md`; `research/iq-sex-differences-verification.md`]

### What I Am Comfortable Saying

- **Older adult meta-analytic claim:** around `d = 0.33` favoring males on `Raven`, but contested. [SOURCE: https://www.sciencedirect.com/science/article/abs/pii/S0160289604000510; `research/iq-sex-differences-test-construction.md`]
- **Representative UK `SPM+` result:** no significant mean or variance sex difference. [SOURCE: https://www.sciencedirect.com/science/article/abs/pii/S0191886911002893; `research/iq-sex-differences-verification.md`]
- **Recent normative `APM` result:** sex not a significant predictor in the cited adult normative paper tracked in this repo. [SOURCE: `research/iq-sex-differences-test-construction.md`]

So the best honest answer is:

- `Raven` can show a **small male advantage**
- but on better representative / latent-style surfaces it can also be **near null**
- therefore `Raven` is cleaner than many verbal or school-loaded batteries, but it is **not** a one-shot proof of a male `g` gap

[INFERENCE]

## Is Raven Timed Or Exhaustable?

It depends on the version.

- **Current commercial `Raven's 2`:** yes, timed and finite. The public `WPS` page gives `30` minutes for ages `4-9` and `45` minutes for ages `9-90`. [SOURCE: https://www.wpspublish.com/ravens2-progressive-matrices-clinical-edition]
- **Classic `SPM` / `APM` literature:** often finite fixed forms with progressively harder items; some administrations are effectively untimed or generous-time power testing, while some research uses explicit short time limits. [SOURCE: https://www.sciencedirect.com/science/article/abs/pii/S0191886911002893; https://www.pearsonassessments.com/en-us/Store/Professional-Assessments/Cognition-%26-Neuro/Raven%E2%80%99s-Advanced-Progressive-Matrices/p/100000414]

So `Raven` is always **form-exhaustable** because it is a finite set of items, but it is **not always clock-exhaustable in the same way** across versions and studies. [INFERENCE]

## Bottom Line

1. No mainstream battery here is literally unlimited.
2. Some batteries are closer to **power testing** than others.
3. Some batteries reward **speed directly**.
4. Some batteries stop because items become **too hard**, not because time ran out.
5. You should never compare scores across batteries as if "same IQ" means "same trait mix."
