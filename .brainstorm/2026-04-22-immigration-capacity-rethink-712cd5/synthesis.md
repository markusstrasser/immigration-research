### Paradigm Gaps

2. pathway and legal-regime composition layers with usable local assignment data
3. a true federal vs state-local welfare timing ledger

### Suggested Next Step
Do not run another county threshold sweep first.

The cheapest high-value sequence is:

1. build a `receiver failure atlas` for `NYC`, `Denver`, `Chicago`, `Boston`, `Bexar`, and one low-drama comparison city
2. compute `resident-weighted exposure` summaries that replace the current median-county newcomer framing
3. write `necessary-condition kill tests` for the strongest remaining claims:
   - open-borders global gains survive only if destination-capacity decay stays below break-even
   - local backlash story survives only if overload proxies beat immigrant-stock proxies
   - resident-harm story survives only if resident-weighted incidence does not disappear after reweighting

If those survive, then plan a convergent pass or `/critique` on one or two survivors. If they fail, stop pretending the county panel is the frontier.
