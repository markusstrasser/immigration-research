# 2026-03-11: Recenter psychometric targeting on the matrix-fluid branch

## Context
The repo's broad current-state docs still center the "general intelligence `g`" question even though the local evidence and the literature audit both say `g` is highly battery-sensitive. The user asked to stop centering `g` and instead focus on the narrower fluid branch.

The fork was not "keep `g`" versus "abandon psychometrics." The real choice was which psychometric target should carry the main load when the repo needs a narrower ability construct.

## Alternatives considered
1. **Keep broad `g` central** — matches the familiar literature framing, but keeps inviting overclaim because the repo's own verification memo says the sex-gap answer changes with battery composition and latent model choice.
2. **Pivot to broad `Gf`** — cleaner than `g`, but still too heterogeneous because `Gf` bundles matrix reasoning, quantitative reasoning, sequence/rule induction, and some spatially loaded tasks with different schooling and design burdens.
3. **Center the matrix / figural / low-knowledge fluid branch** — narrower, closer to the repo's strongest local psychometric evidence, and already operationalized in the matrix-certainty and open-matrix memos.

## Decision
Center the repo's narrow psychometric targeting on the **matrix / figural / low-knowledge fluid-reasoning branch**, not on broad `g`.

This does **not** mean:

- `g` is discarded
- matrix reasoning is a pure natural essence
- the matrix branch alone settles a battery-independent general-ability claim

It means:

- the repo should treat broad `g` as a theory-laden summary whose sex-gap answer is too battery-sensitive to carry the main load
- when a narrower ability surface is needed, the default branch should be matrix / figural / low-knowledge fluid reasoning
- broader `Gf` claims should stay secondary unless the repo freezes a clearer construct map than it has now

## Evidence
- The verification memo says the literature does not establish a single battery-independent sex difference in `g`, while domain-specific surfaces are more stable.
- The matrix-certainty memo already treats the matrix / figural / Raven-like branch as distinct from the full `g` question and says the live uncertainty is on that branch, not on composite pooling.
- The open-data Raven follow-up says the best current local matrix evidence is `ICAR`, where the matrix domain is modestly male-leaning and harder items are more male-leaning without a visible item-reach gap.
- The battery-design memo says Raven-like tests are cleaner than many verbal or school-loaded batteries, but they still do not settle the whole sex-difference question by themselves.

## Revisit if
- a representative, measurement-invariant multi-battery study shows that broad `g` is actually more stable and more informative than the matrix / figural branch once test-design and invariance issues are handled
- the current matrix branch collapses in representative raw Raven-like data or under stronger multi-group latent-item modeling
- the repo freezes a clearer, better-defended broad `Gf` construct that materially improves on the current matrix-fluid branch
